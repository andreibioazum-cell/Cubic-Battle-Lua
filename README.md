1. ОБ ИГРЕ
================================================================================

Cubic Battle 3 - это экшен-игра с видом сверху, где игрок управляет кубиком 
и сражается с врагами в открытом мире. Игра поддерживает моды, сенсорное 
управление, систему скинов и многое другое.

ОСНОВНЫЕ ОСОБЕННОСТИ:
- Динамичный геймплей с врагами
- Поддержка сенсорного управления (мобильные устройства)
- Магазин скинов (обычный и алмазный кубик)
- Полноценная система модов
- Автосохранение прогресса
- Звуковые эффекты
- Встроенная экранная клавиатура

3. УСТАНОВКА И ЗАПУСК
================================================================================

ДЛЯ РАЗРАБОТКИ:
1. Установите LOVE2D версии 11.5+
   Скачать: https://love2d.org/
2. Клонируйте репозиторий
3. Запустите игру: love .

СБОРКА APK (ANDROID):
Используется GitHub Actions (см. build.yml):
1. Пушите код в ветку main или master
2. Actions автоматически соберёт APK
3. Готовый APK будет в артефактах сборки

ТРЕБОВАНИЯ:
- LOVE2D 11.5 или выше
- Для сборки APK: Java 17, Android SDK, NDK 25.2.9519653

КАК СОЗДАТЬ МОД:

-- mods/мой_мод.lua
local мойМод = {}

мойМод.название = "Мой супер мод"
мойМод.автор = "Я"
мойМод.версия = "1.0"

-- Вызывается при загрузке
function мойМод.приЗагрузке()
    print("Мой мод загружен!")
end

-- Вызывается каждый кадр
function мойМод.приОбновлении(дт)
    -- Ваш код здесь
end

-- Вызывается при отрисовке
function мойМод.приОтрисовке()
    love.graphics.print("Мой мод!", 10, 10)
end

-- Вызывается при выстреле
function мойМод.приВыстреле(х, у, дх, ды)
    return { х = х, у = у, дх = дх * 2, ды = ды * 2 }
end

return мойМод

ДОСТУПНЫЕ ХУКИ (СОБЫТИЯ):

Хук                     Когда вызывается              Параметры
-------------------------------------------------------------------------------
приЗагрузке             При загрузке игры             -
приОбновлении           Каждый кадр                   дт (дельта времени)
приОтрисовке            При отрисовке                 -
приВыстреле             Когда игрок стреляет          х, у, дх, ды
приПопадании            Когда враг получает урон      враг, урон
приСмерти               Когда враг умирает            -
приСпавне               Когда появляется враг         враг
приПопаданииПули        Когда пуля попадает           пуля, цель

7. БЕЗОПАСНЫЙ РЕЖИМ
================================================================================

Если игра не запускается из-за мода:
1. Нажмите "Safe Mode" в главном меню
2. Игра автоматически:
   - Создаст резервную копию
   - Удалит все моды
   - Перезапустится

8. СБОРКА ДЛЯ ANDROID
================================================================================

АВТОМАТИЧЕСКАЯ СБОРКА (GITHUB ACTIONS):

name: Build_Cubic_Battle_3
on: [push, pull_request]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout code
        uses: actions/checkout@v4
      - name: Setup Java & Android SDK
        uses: actions/setup-java@v4
      - name: Build APK
        run: ./gradlew assembleEmbedNoRecordDebug

РЕЗУЛЬТАТ СБОРКИ:
- APK файл: CubicBattle3_YYYYMMDD.apk
- game.love: для отладки
- build_info.txt: информация о сборке

9. ОСНОВНЫЕ МЕХАНИКИ
================================================================================

ИГРОК:
- Здоровье: 5 HP
- Скорость: 260 пикселей/сек
- Атака: пули со скоростью 400 пикселей/сек

ВРАГ:
- Здоровье: 5 HP
- Скорость: 150 пикселей/сек
- Атака: стреляет каждые 1.5 секунды
- Награда: 50 монет при смерти

МИР:
- Размер: 3000x3000 пикселей
- Телепортация: нет, есть границы
- Камера: плавно следует за игроком

10. ОТЛАДКА
================================================================================

ЛОГИ:
- Файл install_log.txt - логи установки модов
- Консоль LOVE2D - отладочная информация

КОМАНДЫ РАЗРАБОТЧИКА (в консоли LOVE2D):
modSystem.перезагрузитьМоды()
modSystem.получитьКоличествоМодов()
gameInstaller.получитьСтатусУстановки()

ОШИБКИ:
Если игра выдаёт ошибку "module 'mod_system' not found":
1. Создайте файл mod_system.lua в корневой папке
2. Скопируйте туда код модуля
3. Перезапустите игру
